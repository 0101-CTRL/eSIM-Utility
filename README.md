# eSIM Utility

Standalone FastAPI browser UI for testing Cradlepoint / Ericsson Enterprise APIv3 beta eSIM endpoints.

This utility provides a local browser interface for:

- Viewing eSIM profile inventory
- Creating eSIM profile management tasks
- Uploading bulk eSIM activation CSV files
- Previewing activation jobs before execution
- Executing activation jobs
- Inspecting parent and child activation job status
- Viewing the exact APIv3 URLs generated by the UI

## Endpoint Groups

### `esim_profiles`

Read-only eSIM profile inventory lookup.

Useful for viewing:

- Profile ID
- ICCID
- EID
- Carrier
- Profile name
- Nickname
- Active state
- Classification
- Related net device ID

### `esim_profiles/manage`

Creates profile management tasks.

Supported actions:

- Rename profile nickname
- Set active profile
- Delete profile

The UI includes confirmation guardrails for destructive actions.

### `esim_profile_activations`

Bulk CSV activation workflow.

Supported flow:

- Upload activation CSV
- Preview activation job
- Execute parent activation job
- Inspect parent activation status
- Inspect child activation jobs

## Requirements

Recommended environment:

- Ubuntu, Debian, or Raspberry Pi OS
- Python 3
- Git
- Network access to `https://api.cradlepointecm.com`
- Valid APIv3 bearer token with access to the eSIM beta endpoints

Install system packages:

    sudo apt update
    sudo apt install -y git python3 python3-venv python3-pip

## Install from GitHub

Clone the repository:

    git clone https://github.com/0101-CTRL/eSIM-Utility.git
    cd eSIM-Utility

Create a Python virtual environment:

    python3 -m venv venv

Upgrade pip:

    ./venv/bin/pip install --upgrade pip

Install Python dependencies:

    ./venv/bin/pip install fastapi "uvicorn[standard]" httpx python-multipart

Run the app:

    ./venv/bin/uvicorn app:app --host 0.0.0.0 --port 8011

Open the UI in a browser:

    http://<server-ip>:8011/

Example:

    http://172.55.0.239:8011/

Stop the app with:

    CTRL+C

## Recommended Install Location

For a small lab server or Raspberry Pi, `/opt/api-v3-esim-ui` is a good install location.

Example clean install:

    sudo mkdir -p /opt/api-v3-esim-ui
    sudo chown -R "$USER:$USER" /opt/api-v3-esim-ui
    git clone https://github.com/0101-CTRL/eSIM-Utility.git /opt/api-v3-esim-ui
    cd /opt/api-v3-esim-ui
    python3 -m venv venv
    ./venv/bin/pip install --upgrade pip
    ./venv/bin/pip install fastapi "uvicorn[standard]" httpx python-multipart
    ./venv/bin/uvicorn app:app --host 0.0.0.0 --port 8011

## Optional systemd Service

A sample systemd service file is included here:

    systemd/api-v3-esim-ui.service

The included service assumes:

- App path: `/opt/api-v3-esim-ui`
- Linux user: `ubuntu`
- Port: `8011`

To install the service on a system using the `ubuntu` user:

    cd /opt/api-v3-esim-ui
    sudo cp systemd/api-v3-esim-ui.service /etc/systemd/system/api-v3-esim-ui.service
    sudo systemctl daemon-reload
    sudo systemctl enable --now api-v3-esim-ui
    sudo systemctl status api-v3-esim-ui --no-pager

If your Linux username is not `ubuntu`, edit the service before starting it:

    sudo nano /etc/systemd/system/api-v3-esim-ui.service

Change these lines:

    User=ubuntu
    Group=ubuntu

For example:

    User=myuser
    Group=myuser

Then reload and restart:

    sudo systemctl daemon-reload
    sudo systemctl restart api-v3-esim-ui

## Basic Usage

1. Open the UI.
2. Paste an APIv3 bearer token into the token field and click Add Token to verify access.
3. Use `eSIM Profiles` to look up profiles.
4. Use `Manage Profile` to preview and submit profile management tasks.
5. Use `Bulk Activations` to upload a CSV and create a preview job.
6. Review the returned parent job ID.
7. Execute only after confirming the preview looks correct.
8. Inspect parent and child jobs for status details.

## CSV Format for Bulk Activation

The bulk activation CSV requires these columns:

    eid,activation_string

Optional columns:

    confirmation_code,nickname

The UI includes a CSV Format Help panel showing the required and optional columns, plus a Download Sample CSV button.

## Security / Lab Use Warning

This tool is intended for local lab testing only. Do not expose the service publicly.

Bearer tokens are entered in the browser UI and sent to the local FastAPI proxy for each API request. The app does not intentionally persist tokens, but the service should still be treated as sensitive because it can perform real eSIM management actions.

Avoid running this service on a public internet-facing interface unless it is protected by firewall rules, VPN access, or another trusted access-control layer.

## Troubleshooting

### Port already in use

Check what is using port 8011:

    sudo ss -ltnp | grep 8011

### Service not found

Create/install the systemd service first:

    sudo cp systemd/api-v3-esim-ui.service /etc/systemd/system/api-v3-esim-ui.service
    sudo systemctl daemon-reload
    sudo systemctl enable --now api-v3-esim-ui

### Python import or dependency error

Reinstall dependencies:

    cd /opt/api-v3-esim-ui
    ./venv/bin/pip install --upgrade pip
    ./venv/bin/pip install fastapi "uvicorn[standard]" httpx python-multipart

### Syntax check

Run:

    cd /opt/api-v3-esim-ui
    ./venv/bin/python -m py_compile app.py

### Restart service

Run:

    sudo systemctl restart api-v3-esim-ui
